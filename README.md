## How to run ?

### Create key.txt file
```
[DISCORD BOT TOKEN]
[OPENAI_API_KEY]
```
### Set chunk_size
You can adjust the 'chunk_size' and 'chunk_overlap' as needed.
The default values are 1000 and 500, respectively.
